Bienvenue sur Mon Serveur Node.js

Introduction

Ce serveur est configuré pour servir une application web développée avec Vue.Js.

Technologies Utilisées

    Node.js pour le côté serveur

    Express.js pour la gestion des routes

    Mysql et PHPMyAdmin

    HTML, CSS, JavaScript pour le frontend

Projets Hébergés

Le serveur Node.js exécute les différentes requêtes pour l'application web directement sur une base de données située sur PHPMyAdmin.

/followers

La commande followers permets API de faire une requete sql dans la table followers
